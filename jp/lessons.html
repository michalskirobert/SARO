UNIT 0

アルファベット

チェシチ！
このレッスンページでは、ポーランド語の読み方と、ポーランド語で使用される文字と単語の発音の仕方を学んでいきます。

ポーランド語の文字は、アルファベットが基礎となった全32文字を使用します。この文字は一般的に【abecadło(アベツァドゥウォ)】と呼ばれておりますが、正式な名称ではないようです。この【abecadło(アベツァドゥウォ)】という言葉は、ユリアン・トゥヴィムというポーランド人の詩人が書いた詩に同じ題名のものがあるため、小さい子どもたちにはとても有名です。

ポーランド語の発音はあまり難しくありません。ほとんどの発音が、日本語や韓国語・中国語と近いものがあります。皆さまには、最も簡単な発音方法をお教えしていきます。

もちろん、単語によっては文字だけとは異なる読み方になります。

ポーランド語には2つの読み方のグループが存在します。一つ目は【samogłoski(サモグウォスキ) : 母音】と呼ばれるもので、そのまま発音が可能な音です。二つ目は【spółgłoski(スポブウォスキ) : 子音】と呼ばれるもので、母音のグループに属する音と一緒でなければ発音が難しい音です。例として、以下のような発音になります。
母音なしの場合 : b, p, t, d, s, h, k
母音ありの場合 : be, pe, te, de, sy, ha, ka

(jot, igrek, zet, ziet, żet)

ポーランド語には二重文字と三重文字が存在します。また、これらはアルファベットの一部ではありませんが、重要な事なので覚えておくと便利です。

Ch – H のような音 例: chałwa
Ci – Ć の音＋少し長めのI のようの音 例: cisza
Cz – 英語のch のような音 例: czarny
Dz -C と Z の間のような音 例 dzban
Dzi - ci と ź の間のような音 例: dzik
Dź – dziを短めに出す音 例: dźwięk
Ni -Ń を長めに出す音 例: nic
Rz – Ż の音 例: rzeka
Si – Ś を長めに出す音 例: siano
Sz – 英語の sh の音 例: szok
Zi – Ź を長めに出す音 例: zima

ポーランド語のいくつかの単語は、他の国の言語が由来となっています。そのため、次に紹介する文字は公式のアルファベットではありませんが、発音は以下のようになります。

X - [iks] - 例: eks, ksylofon
Q - [ku] - 例: kwartet, quad
V - [w] - 例: wentylacja
今紹介したそれぞれの単語は別の綴りで書く事がありますが、この綴りがオリジナルになります。別の綴りは以下のようになります。
例 : ex, xylophone, quartet, quad, ventilation.
また、Ｖを単独で綴る場合も例外です。

ポーランド語には、単語の書き方に関して一連のルールがあり、文章によって意味が変わる場合があります。例えば、【morze : 海】と【może : 恐らく】は同じ発音になります。綴りについては別のページでさらに詳しく紹介されていますので、ぜひ確認してみて下さい。もし綴りについては難しくて苦手だという方は、単語の読み書きだけ覚えてみて下さい。

語形変化

ポーランド語の単語は、たとえ簡単な文章であっても、状況によって多少異なる場合があります。 例を見てみましょう：


《基本形》

Dom - 家

《文章に組み込まれた場合》

Idę do domu. – 私は家に向かっています(私は家に向かいます)

さて、どうしてでしょうか？ それはポーランド語には、「語形変化」と呼ばれる特性があるからです。 簡単に説明すると、名詞が文での役割に応じて形を変えることで、 動詞の変化を表す活用と似ています。 語形は文脈によって決まる場合もあれば、隣接する前置詞や助詞によって決まる場合もあります。 この語形変化は、言語学者によっては「ケース」と呼ばれることがあり、このレッスン内でも「ケース」という言葉を使用して説明していきます。

語形変化はポーランド語において非常に重要なパートですが、ご心配なく◎ネイティブであるポーランド人でさえ、正しいケースを選択するのが難しい場合があります。

ポーランド語の語形変化は「deklinacja(デクリナツヤ：語形変化)」、よりカジュアルに言う場合は「odmiana przez przypadki(オドゥミアナ プシェズ プシェパドゥキ：ケースを介しての変化」と呼ばれます。 この文脈での「Przypadek」は、名詞の単一のケースを意味します。 その複数形は「przypadki」です。  

ポーランド語には7つのケースがあり、これらのケースにはそれぞれに関連する疑問形があります。  

Mianownik(主格)
kto?(クト：誰)  co?(ツォ：何)

Dopełniacz (所有格)
kogo?(コゴ：誰)  czego?(チェゴ：何)

Celownik(与格)
komu?(コム：誰に)  czemu?(ツェム：なぜ)

Biernik(対格)
kogo? (コゴ：誰)  co?(ツォ：何)
Narzędnik(助格)
z kim? (ズ キム：誰と)  Z czym?(ズ チェム：何と)

Miejscownik(所格)
o kim? (オ キム：誰について) 
o czym?(オ チェム：何について)

Wołacz(呼格) o!(オ：オー！)


ここからは、各ケースを一つずつ説明していきます。


Mianownik(主格):

最も基本的な形です。 疑問形はkto？ co？ （ベーシックな「誰ですか？」「何ですか？」）


Co to jest? (ツォ ト イェストゥ？)
To jest mój telefon. (ト イェストゥ モイ テレフォン)
《訳》
これは何ですか？
これは私の電話です。


Kto jest na tym zdjęciu?　(クト イェストゥ ナ テム ズイェンチゥ？)
Znajomy ze szkoły.　(ズナヨミ ゼ シュコウィ)
《訳》
写真に写っているのは誰ですか？
学校の友達です。


Dopełniacz(所有格):

このケースは、何かが欠けていたり、何かを必要としたり望んでいることを示します。 疑問形は：kogo？Czego?


Czego nie wziąłeś? (チェゴ ニエ ヴジョエシ？)
Mojego telefonu. (モエゴ テレフォヌ)
《訳》
何を持っていかなかったのですか？
私の電話です。


Kogo dzisiaj nie ma? (コゴ ジシェ ニエ マ？) Mojego znajomego. (モエゴ ズナヨメゴ) 《訳》 今日は誰がいないですか？ 私の友達です。


Celownik(与格):

このケースは、行動のターゲットを示します。例えば、誰かに物事を与えるときです。
疑問形はkomu？ Czemu?


Komu dajesz prezent? (コム ダイェシュ プレゼントゥ？)
Mamie. (マミエ)
《訳》
誰にプレゼントをあげたのですか？
お母さんです。



Czemu się przyglądasz? (チェム シエン プシェグロンダシュ？)
Obrazowi na ścianie. (オブラゾビ ナ シチャニエ)
《訳》
何を見ているのですか？
絵が描かれている壁です。

Biernik(対格)

このケースもまた、行動のターゲットを示します。 読書、食事、買い物などに関する行動を起こす際によく使用されます。
疑問形はkogo？ co？

Biernik(対格)、Celownik(与格)、もしくは他のケースを使用するかどうかは、ほとんどの場合、前置詞と動詞によって異なります。一般に、BiernikはCelownikよりも頻繁に使用されるため、何を使用するかわからない場合は、Biernikを使うと良いでしょう。

Co czytasz? (ツォ チタシュ？)
Artykuł w wiadomościach. (アルティクフ ヴ ヴィアドモシチャ)
《訳》
何を読んでいるのですか？
ニュースに関する記事です

Kogo zapraszasz? (コゴ ザプラシャシュ？)
Mojego chłopaka. (モイェゴ クウォパカ)
《訳》
誰を招待しますか？
私の彼氏です。


Narzędnik(助格)

このケースは、行動(アクション)が目的を実行する手段として機能する場合に使用されます（「narzędzie」とはポーランド語で「手段/方法」を意味します）。 また、特定の前置詞（例：z-with、przed-before）とともに使用されます。 疑問形は z kim? z czym? (特定の内容で kim? czym?)

Czym jedziecie na wakacje? (チェム イェジェチエ ナ ヴァカチェ？)
Samochodem (サモホデム)
《訳》
交通手段は何ですか？(※直訳：休暇は何をしますか？)
車です。

Z kim rozmawiasz? (ズ キム ロズミヴィアシュ？)
Z żoną. (ズ ジョノ)
《訳》
誰と話しているのですか？
私の妻です。  

Miejscownik(所格)

前置詞と共に起こるケースで、例えばw - in, into, na - on, ontoと一緒に使用します（「miejsce」はポーランド語で「場所」を意味します）。また、前置詞o（訳：約）と一緒に使用してトピックを示します。 疑問形はo kim? o czym?となり、特定の内容ではkim? czym?となります。

O kim rozmawiasz? (オ キム ロズマヴィアシュ？)
O prezydencie. (オ プレジデンチェ)
《訳》
何を話しているのですか？
大統領についてです。


Gdzie jest jedzenie? (ジェ イェストゥ イェジェニエ？)
Na stole. (ナ ストレ)
《訳》
食べ物はどこにありますか？
テーブルの上です。

Wołacz(呼格)

めったに使用されないケースですが、ある種の状況でのみ使用されます。例えば、誰かに直接呼びかける時や、ある事実を強調したい時（大勢の人に話しかける時）、また、スピーチを行う際などに使用されます。

Mamo! Przyjdź do mnie! (マモ！プシェイジ ド ムニエ)
《訳》
ママ！こっちに来て！

A ty co sądzisz, Moniko?
《訳》
モニカはどう思いますか？
(※モニカ＝”Monika” ですが、「wołacz(呼格)」を使用すると”Moniko” となります。)

さて、今まで紹介してきたものがポーランド語の語形変化の全部となります。たくさんあるように思えるかもしれませんが、ポーランド語で旅を始めるときは、考えすぎないでください。 勉強を始めるうえで最も重要なのは、すべてのケースを暗記することではなく、自分で例文を作り、理解を深めることです。あなたが普段直面する状況・場面を思い出し、特定の動詞、構造を考えて、例文を作ってみてください。 すべてのケースを知ろうとしたり、すべての単語を覚えなきゃと思わないことがキーです◎ 実際、存在する単語は思いがけないほどたくさんありますが、少なくとも50％は未使用のままです。  

ご参考までに、語形変化を勉強する際によく目にする変化表のサンプルのご紹介しておきます。

新しい単語と文法を勉強したら、ぜひまたここに戻ってきて下さい♪ より理解が深まるはずです。一緒に頑張って、ポーランド語をマスターしましょう！

語順

言葉を話すときに最も重要なことは、文章を作ることです。単語をたくさん覚えることは出来ても、それらを組み合わせる能力は自身のスキルを直接示すものとなります。このレッスンでは、ポーランド語の基本的な文章の作り方について簡単に説明します。
まずは、語順の説明をしていきます。 単語を並べる順番は以下の通りです。

主語＋動詞＋目的語 (英語でいうSVO)

分かりましたか？まず一つ目に、 《誰》 が行動したか、二つ目に彼らが 《何をした》 か、そして次に 《誰と》《どのツールを使用して》《どこで》という順番になります。基本的には英語と同じ語順と考えて大丈夫です◎


例文:

Monika（ モニカ ） czyta（ チェタ ） książkę（ クションシュケン ）
モニカは本を読んでいます。 (モニカは本を読みます。)

Moja mama（ モヤ ママ ） pracuje w banku（ プラツイェ ヴ バンク ）
私の母は銀行で働いています。

James（ ジェイムス ） nie（ ニエ ） mówi（ モヴィ ） po（ ポ ） polsku.（ ポルスク ）
ジェームスはポーランド語を話しません。
また、強調する言葉によって、語順や追加の単語が少し変わる場合があります。 ノーマルな文章では、最後の言葉が最も強調されます。
次の例文を見てみましょう。


Pierogi（ ピエロギ ） gotuje（ ゴトゥイェ ） bardzo（ バルズォ ） często.（ チェンスト ）
（私はピエロギを頻繁に作ります！）
Bardzo（ バルズォ ） często.（ チェンスト ） gotuje（ ゴトゥイェ ） pierogi（ ピエロギ ）
（ピエロギは他の食べ物と比べるとよく作ります。）


ここで余談ですが、複雑な文では、1つの文に複数の正しい語順があります。まずは、動詞の前には主語が来ることだけ覚えておいてください！
時間を示す単語は多くの場合、文頭にその単語を置きます。

Jutro（ ユトゥロ ） nie（ ニエ ） idę（ イデ ） do（ ド ） pracy（ プラツェ ）
私は明日仕事に行きません。

Teraz mieszkam w Krakowie （ テラス ミェシュカム ヴ クラコヴィエ ）
私は今クラクフに住んでいます。

否定を示したい場合には、動詞の前に 《nie》を追加するだけで完成します。
ポーランド語の動詞は活用させる必要があるため、実際に主語を使用しなくても主語が何かを推測することができます。 英語とは異なり、主語となるものがすでに説明されている場合には、主語を省略して文を作ることができます。 これは、会話でも文面でも同じです。

例文:

私は本を読んでいます。 →

Ja czytam książkę （ ヤ チェタム クションシュケン ）

→

Czytam książkę （ チェタム クションシュケン ）

私には兄弟がいません。 → Ja nie mam rodzeństwa（ ヤ ニエ マム ロゼニストゥヴァ ） → Nie mam rodzeństwa（ ニエ マム ロゼニストゥヴァ ）

彼女はリンゴを食べています。 → Ona je jabłko. （ オナ イェ ヤブコ ） → Je jabłko. （ イェ ヤブコ ） (※文脈から主語が読み取れた場合)

彼らは家に帰ります。 → Oni idą do domu（ オニ イドン ド ドム ） → Idą do domu（ イドン ド ドム ） (※文脈から主語が読み取れた場合)


質問をする:

質問文を作るのに、難しい文法はありませんのでご安心下さい◎ 文の最初に《czy》を追加することで、はい/いいえ で答えられる質問文が完成します。

Czy mówisz po polsku?（ チュ モヴィシュ ポ ポルスク ） / Mówisz po polsku?（ モヴィシュ ポ ポルスク ）
ポーランド語を話しますか？

Czy czytasz książkę? （ チュ チュタシュ クションシュケン ） / Czytasz książkę? （ チュタシュ クションシュケン ）
本を読んでいるのですか？(※2つ目の方がより伝わりやすい)

Czy ona ma rodzeństwo?（ チュ オナ マ ロゼニストゥヴォ ） / Ona ma rodzeństwo?（ オナ マ ロゼニストゥヴォ ）
彼女は兄弟がいますか？
(※1つ目の文の方が丁寧な質問の仕方)

形容詞

語形変化のレッスンで、ポーランド語の名詞はケース（przypadki）に応じて変化するという事を学びました。実は形容詞も同じです。形容詞にも7つのケースがあり、正しい変化(odmiana)を見つけるために、私たちが尋ねる質問は変えられません。
形容詞の働きは名詞を修飾すること、すなわち《何（何の種類の）？/どれ？（ポーランド語：jaki, jaka, jakie?）》、《どれ（どっち）？（ポーランド語：który, która, które?）》、《誰？ （ポーランド語：czyj, czyja, czyje?）》などの質問に答える事です。

こちらが例です：

形容詞は、会話の中で変化することができます。名詞と同じ規則に従って、ケース、数、性別によって変化します。形容詞の最後の文字(これより先、「末尾」という)を正しく変換させるためには、何のケースか、説明したい名詞の文法的な性別は何か、名詞が複数形か単数形かを知る必要があります。
ほとんどの形容詞は、次のような基本的な規則に従って変化します:

ピンク 主格の女性形容詞の基本形
グリーン ・末尾が「e」で終わる
・最後のスペルが【 ka (例：wielka), ga (例：długa), ia (例：głupia) 】で終わる主格の女性形容詞の場合→末尾が「i」に変わる
・その他の場合→「i」なし
黄色 ・末尾が「i」「y」で終わる
・最後のスペルが【 ka (例：wielka), ga (例：długa), la (例：ośla), ia (例：głupia) 】で終わる主格の女性形容詞の場合→末尾が「i」に変わる
・その他の場合→「i」なし
青 ・最後のスペルが【 ia (例：głupia) 】で終わる主格の女性形容詞の場合→末尾が「i」に変わる
・その他の場合→「i」なし
赤 ・「i」または「y」を追加して、直前の音を柔らかく(軟子音化)する必要がある
注意：末尾が-szyまたは-ży（※-rzyではない）は、音声的に-si、-ziのように柔らかい音(軟子音)で表現される場合があります。

例：

ten wyższy – ci wyżsi
ten głuchy – ci głusi
ten duży – ci duzi

*ですがこちらは：
en boży → ci boży
ten stary → ci starzy
ten kurzy → ci kurzy

こちらは主格の男性形容詞(単数形)の軟子音の例です。

名詞の文法的な性別

ポーランド語の勉強をしている皆さんは、名詞には性別があるということを知っている方が多いと思います。このレッスンでは、何がどのように違うかをご紹介していきます。ではまず、次の例文をご覧ください。

Damian( ダミアン ) : Cześć! Jestem Damian( チェシチ　イェステン　ダミアン ) , mam 28 lat( マン ドワジェシチア　ヲシエン　ラツ ) . Urodziłem się w Polsce( ウロジウェン　シエ　フ　ポルスツェ ) , jestem Polakiem ( 　イェステン　ポラキエン ) . Pracuję jako ( プラツイェ　ヤコ ) 　nauczyciel w szkole.( ナウチツェル　フ　シコレ )

Anna( アンナ ) : Cześć! Jestem Anna( チェシチ　イェステン　アンナ ) , mam 27 lat.( マン　ドワジェシチア　シエデム　ラツ ) Urodziłam się w Polsce( ウロジワン　シエ　フ　ポルスツェ ) , jestem Polką ( 　イェステン　ポルコン　 ) . Pracuję jako ( プラツイェ　ヤコ ) 　nauczycielka w szkole.( ナウチツェルカ フ　シコレ )

これを訳すと以下のような文になります。

Damian: こんにちは！私はDamianといいます。年は28歳で、ポーランド出身のポーランド人です。私は学校で教師として働いています。

Anna: こんにちは！私はAnnaといいます。年は27歳で、ポーランド出身のポーランド人です。私は学校で教師として働いています。

格は別として、文章を作っていく段階で重要な役割を果たしているものは性別です。上記の例文の場合、両者の間で完全に異なっているものは話している人の「性別」です。ご覧の通り、3か所の赤文字になっている部分は、話している人の性別によって変化しています。
ポーランド語の文法を学んでいくうえで、文法的な性別はとても重要な要素です。では、さっそく名詞の性別についてより詳しいことを学んでいきましょう。
ポーランド語の全ての名詞には性別があります。3つの基本的な性別が以下の通りです。：

  • 男性名詞 (rodzaj męski( ロジャイ　メンスキ ) )
  • 女性名詞 (rodzaj żeński( ロジャイ　ジェンスキ ) )
  • 中性名詞 (rodzaj nijaki( ロジャイ　ニヤキ ) )

まずは人を示す名詞からご紹介していきます。性別を区別するのはとても簡単です
Ten mężczyzna( テン　メンシチズナ ) – この男性 → 男性名詞
Ten chłopiec( テン　フヲピエツ ) – この男の子 → 男性名詞
Ta kobieta( タ　コビエタ ) – この女性 → 女性名詞
Ta dziewczyna( タ　ジェフチナ ) – この女の子 → 女性名詞
To dziecko( ト　ジェツコ ) – この子供 → 中性名詞
どうですか？簡単ですよね◎

では、次にモノや動物に関する名詞の性別について勉強していきましょう。モノや動物の場合、文法的な性別は実際の性別とは何も関係がありません。例えば、窓やリンゴの実際の性別など誰も分かりませんよね。ポーランド語の名詞を勉強するときは、性別とセットで覚えることを意識しましょう♪

男性名詞の例：

dom( ドン ) – 家 ten dom( テン　ドン ) – この家
samochód( サモフド ) – 車 ten samochód( テン　サモフド ) - この車
pies( ピエス ) - 犬 ten pies( テン　ピエス ) - この犬
kot( コト ) - 猫 ten kot( テン　コト ) - この猫
deszcz( デシチ ) - 雨 ten deszcz( テン　デシチ ) - この雨
las( ラス ) - 森 ten las( テン　ラス ) - この森
długopis( ヅウゴピス ) - ペン ten długopis( テン　ヅウゴピス ) - このペン

女性名詞の例:

kamera( カメラ ) – ビデオカメラ ta kamera( タ　カメラ ) – このビデオカメラ
gitara( ギタラ ) – ギター ta gitara( タ　ギタラ ) - このギター
szkoła( シコワ ) - 学校 ta szkoła( タ　シコワ ) - この学校
butelka( ブテルカ ) - ボトル ta butelka( タ　ブテルカ ) - このボトル
herbata( ) - お茶 ta herbata( タ　ヘルバタ ) - このお茶
kawa( カワ ) - コーヒー ta kawa( タ　カワ ) - このコーヒー
krowa( クロワ ) - 牛 ta krowa( タ　クロワ ) - この牛

基本的に「-a」で終わる名詞は【女性名詞】です。ただし、いくつか例外もあるので注意しましょう。（例：sprzedawca( スプシェダフツァ ) – 店員 ／ kierowca( キエロフツァ ) – 運転手…これらは【男性名詞】）

また、女性名詞でも「-a」で終わらない珍しいケースもあります。（例：sól( スル ) – しお）


中性名詞の例：

okno( オクノ ) - 窓 to okno( ト　オクノ ) - この窓
oko( オコ ) - 目 to oko( ト　オコ ) - この目
bagno( バグノ ) - 沼地 to bagno( ト　バグノ ) - この沼地
piwo( ピヲ ) - ビール to piwo( ト　ピヲ ) - このビール
auto( アウト ) - 車 to auto( ト　アウト ) - この車
ciasto( チャスト ) - ケーキ to ciasto( ト　チャスト ) - このケーキ
kakao( カカオ ) - ココア to kakao( ト　カカオ ) - このココア

ご覧の通り、基本的に中性名詞は「-o」で終わります。

以上が、名詞の性別に関する基礎知識です。最初は分からなくても、すぐに理解できるようになります◎焦らずゆっくり、自分のペースで理解を深めていきましょう。名詞を覚える時は、文法的な性別と合わせて覚えることを習慣にしましょう。
